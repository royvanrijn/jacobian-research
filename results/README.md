# Result artifacts

This directory separates retained proof inputs from regenerated experiment
output.

## Retained

- `certificates/`: characteristic-zero msolve inputs and their matching reduced
  bases. Most outputs are `[1]` certificates; the `(9,27)` file is an exact
  elimination certificate. Run `make verify-certificates` to reproduce them.
- `systems/`: modular F4SAT inputs for the two reduced `(72,108)` polygons.
  Their last polynomial is a vertex saturator, so invoke msolve with `-S`.
- `OBSTRUCTION_CLUSTERS.md`: compact cross-stage obstruction summary.
- `SCAN_SUMMARY.md`: exact scope and outcome of the degree 5--20 pilot.

## Generated locally

Top-level `*.json` and `*.txt` files are ignored by Git. They include detailed
scan records and the large sparse 95D/510D counterexample artifacts. Each is
regenerated by the correspondingly named script in `scripts/`; the root README
contains the reproduction commands.

Solver timeouts in generated JSON are complexity measurements, not proof
certificates or candidate solutions.
