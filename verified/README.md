# Verified core: C01--C04

This directory contains the stable proof chain:

- [C01](FOUNDATIONAL_GEOMETRY.md): exact determinant and collision;
- [C02](MARKED_ROOT_MODEL.md): the cubic marked-root isomorphism;
- [C03](IMAGE_AND_NONPROPERNESS.md): exact image, fibers, and nonproperness;
- [C04](WEIGHTED_SEED_THEOREM.md): weighted construction and symmetric
  monodromy.

The external Lean certificate for C01 remains in [LEAN_C01.md](LEAN_C01.md).
Expanded audit and normalization narratives are preserved under
[archive/core-support](../archive/core-support/README.md), while their commands
remain in the public [reproduction guide](../REPRODUCE.md).

Start with [FOUNDATIONAL_GEOMETRY.md](FOUNDATIONAL_GEOMETRY.md). The paper
version is being assembled under
[papers/core-counterexample](../papers/core-counterexample/main.tex).

These four documents are the primary core references.
