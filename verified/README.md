# Verified core: C01--C04

This directory contains the stable proof chain:

- C01: explicit determinant and collision;
- C02: the cubic marked-root isomorphism;
- C03: exact image, fibers, and nonproperness;
- C04: the weighted marked-root construction and symmetric monodromy.

Start with [FOUNDATIONAL_GEOMETRY.md](FOUNDATIONAL_GEOMETRY.md). The paper
version is being assembled under
[papers/core-counterexample](../papers/core-counterexample/main.tex).

Theorems here are frozen except for editorial corrections and independent
verification work.
