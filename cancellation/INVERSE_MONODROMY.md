# Monodromy of the cancellation inverse polynomial

For positive integers `m,r`, consider

\[
 \Psi(T)=C\int_0^T\bigl(1-t(Q-Pt)^m\bigr)^r\,dt-R,
 \qquad C P Q\ne0.
\]

Its degree is

\[
 N=r(m+1)+1,
\]

and its derivative has `m+1` roots, each with multiplicity `r`, on the
generic parameter locus.  This note records a proof draft of the resulting
generic monodromy theorem and the numerical branch-cycle search used to test
it.

## 1. Affine normalization

Put

\[
 x=\frac{P}{Q}T,
 \qquad
 \lambda=\frac{Q^{m+1}}P.
\]

Affine changes in source and target do not change monodromy, so it is enough
to study

\[
 F_\lambda(x)=\int_0^x
 \bigl(1-\lambda u(1-u)^m\bigr)^r\,du.                 \tag{1}
\]

Write

\[
 g_\lambda(x)=1-\lambda x(1-x)^m.
\]

The critical points fail to be distinct only when

\[
 \lambda=\lambda_*=\frac{(m+1)^{m+1}}{m^m}.           \tag{2}
\]

Indeed,

\[
 g_\lambda'(x)=-\lambda(1-x)^{m-1}(1-(m+1)x),
\]

and a common zero of `g_lambda` and `g_lambda'` must have
`x=1/(m+1)`, which gives (2).

## 2. Generic critical-value separation

It remains to rule out the possibility that two distinct critical points
always have the same critical value.  This can be done directly at large
`lambda`.

As `lambda` tends to positive infinity, one critical point satisfies

\[
 c_0=\lambda^{-1}+O(\lambda^{-2}),                    \tag{3}
\]

and the other `m` critical points are indexed by the `m`-th roots of unity:

\[
 c_\zeta=1-\zeta\lambda^{-1/m}+o(\lambda^{-1/m}),
 \qquad \zeta^m=1.                                    \tag{4}
\]

The first critical value is `F_lambda(c_0)=O(lambda^(-1))`.  On the other
hand,

\[
 F_\lambda(1)=
 \sum_{j=0}^r(-\lambda)^j\binom rj
 \operatorname{B}(j+1,mj+1),                         \tag{5}
\]

whose leading coefficient is nonzero and whose degree in `lambda` is `r`.
For the tail from `c_zeta` to one, put
`y=1-u=lambda^(-1/m)z`.  Then

\[
 \begin{aligned}
 \int_{c_\zeta}^1 g_\lambda(u)^r\,du
 &=\lambda^{-1/m}
   \int_0^{\zeta+o(1)}
   \bigl(1-(1-\lambda^{-1/m}z)z^m\bigr)^r\,dz\\
 &=A_{m,r}\,\zeta\lambda^{-1/m}+o(\lambda^{-1/m}),  \tag{6}
 \end{aligned}
\]

where

\[
 A_{m,r}=\int_0^1(1-s^m)^r\,ds>0.
\]

Consequently

\[
 F_\lambda(c_\zeta)
 =F_\lambda(1)-A_{m,r}\zeta\lambda^{-1/m}
  +o(\lambda^{-1/m}).                                 \tag{7}
\]

The `m` values in (7) are pairwise distinct for all sufficiently large
`lambda`; by (5), they are also distinct from the value at `c_0`.  Thus the
critical values of (1) are distinct on a nonempty Zariski-open subset of the
`lambda`-line.

It follows that the generic finite branch cycles are `m+1` individual
`(r+1)`-cycles.  Infinity contributes an `N`-cycle.

## 3. The hypertree group lemma

The group-theoretic input is independent of the cancellation formula.

**Lemma.**  Let `s>=2` and `k>=2`, and let a transitive permutation group on

\[
 n=1+s(k-1)
\]

letters be generated by `s` individual `k`-cycles.  Then it is `A_n` when
`k` is odd and `S_n` when `k` is even.

**Proof.**  Form the bipartite incidence graph whose vertices are the `n`
letters and the `s` cycle supports, with an edge for each incidence.  The
graph is connected because the generated group is transitive.  It has

\[
 n+s=1+sk
\]

vertices and exactly `sk` edges, hence it is a tree.

Root this incidence tree at one cycle support.  The second support meets the
first in exactly one letter, and every later support can be ordered so that
it meets the union of the previous supports in exactly one letter.

Suppose first that

\[
 \alpha=(a_1,\ldots,a_k),
 \qquad
 \beta=(a_1,b_2,\ldots,b_k)
\]

have supports meeting only at `a_1`.  Conjugating `beta` by powers of
`alpha` gives

\[
 \beta_i=(a_i,b_2,\ldots,b_k).
\]

For `i!=j`, the product `beta_i beta_j^(-1)` is a 3-cycle on
`{a_i,a_j,b_2}`.  These 3-cycles generate the alternating group on
`{a_1,...,a_k,b_2}`.  Conjugation by `beta` then successively supplies
3-cycles containing `b_3,...,b_k`, so the two original cycles generate a
group containing `A_(2k-1)`.

Now adjoin the remaining supports in rooted-tree order.  At each step the
new `k`-cycle meets the old letters at one point.  Conjugating it by the
alternating group already obtained moves that attachment point arbitrarily;
the same 3-cycle argument adds all `k-1` new letters.  Induction gives
containment of `A_n`.

If `k` is odd, all generating `k`-cycles are even, so the group is exactly
`A_n`.  If `k` is even, a generator is odd, so the group is `S_n`.  This
proves the lemma.

## 4. Generic monodromy theorem

Apply the lemma with

\[
 s=m+1,
 \qquad k=r+1,
 \qquad n=1+(m+1)r=N.
\]

The cover defined by a polynomial is connected, so its geometric monodromy
is transitive.  The branch cycle at infinity is the inverse of the product
of the finite branch cycles, so the finite cycles already generate this
transitive group.  Sections 1 and 2 give precisely `s` finite `k`-cycles on
the generic locus.  Therefore

\[
 \boxed{
 G_{m,r}=\begin{cases}
 A_N,&r\text{ even},\\
 S_N,&r\text{ odd}.
 \end{cases}}                                          \tag{8}
\]

There are no imprimitive or exceptional generic passports for positive
`m,r`.  Special parameter values where critical points or critical values
collide can of course have different specialized passports; (8) concerns
the generic cancellation family.

## 5. Computational search

The script

[`search_cancellation_inverse_monodromy.py`](../scripts/search_cancellation_inverse_monodromy.py)

uses the rational specialization `lambda=4(m+1)`.  Before continuation it
checks exactly over `QQ` that

1. `g_lambda` is separable; and
2. the critical-value resultant has degree `m+1` and is separable.

It then lifts roots around explicit lollipop loops, requires every local
permutation to have cycle type `(r+1)`, and computes transitivity,
primitivity, and group order by Schreier--Sims.  Every tuple is also emitted
in GAP syntax.

The search was run for

\[
 1\le m\le6,
 \qquad 1\le r\le5,
 \qquad 3\le N\le36.                                  \tag{9}
\]

All 30 cases were transitive and primitive and gave exactly (8).  This grid
contains composite degrees, composite local-cycle lengths, and all parity
combinations in the stated range.  Representative machine-readable tuples
are retained in

- [`cancellation_inverse_monodromy_small.json`](../artifacts/cancellation_inverse_monodromy_small.json),
- [`cancellation_inverse_monodromy_6_4.json`](../artifacts/cancellation_inverse_monodromy_6_4.json), and
- [`cancellation_inverse_monodromy_6_5.json`](../artifacts/cancellation_inverse_monodromy_6_5.json).

The bounded experiment is evidence and a debugging aid, not the proof of
(8); the proof is the critical-value separation plus the hypertree lemma.
The stored tuples are replayed using exact permutation arithmetic by

[`verify_cancellation_inverse_monodromy_artifacts.py`](../scripts/verify_cancellation_inverse_monodromy_artifacts.py).

To rerun a single case or the full displayed grid:

```bash
.venv/bin/python scripts/search_cancellation_inverse_monodromy.py 6 4
.venv/bin/python scripts/search_cancellation_inverse_monodromy.py \
  --grid 1:6 1:5 --json artifacts/cancellation_inverse_monodromy_6x5.json
.venv/bin/python scripts/verify_cancellation_inverse_monodromy_artifacts.py
```

The local `sage` launcher was unavailable during the initial run and no
standalone GAP executable was installed.  The search therefore uses only the
repository's pinned SymPy/mpmath environment, while printing exact integer
permutations in GAP syntax for independent replay.

## 6. Arithmetic consequence to develop next

Equation (8) supplies the geometric deck group needed for a cancellation-
family Chebotarev theorem.  The remaining arithmetic work is not another
group classification: it is to choose an integral model, extract an explicit
bad-prime ideal from the leading coefficient and branch-value discriminant,
and compare geometric with arithmetic monodromy after reduction.  This is
independent of the truncated-binomial parameter polynomial `M_(m,r)(q)`.
