# Archived cancellation component notes

This directory preserves the detailed narratives that were formerly grouped
under the single C24 umbrella.  They include separate notes on boundary
exhaustion and auditing, thick intersections, invariant layers, arithmetic,
monodromy, target automorphisms, and several rigidity relaxations.

They are retained for derivations and historical reference, but they are not
the active reading path and should not be cited as separate headline claims.
The consolidated programme is:

- [construction](../../experimental/cancellation/CONSTRUCTION.md);
- [arithmetic](../../experimental/cancellation/ARITHMETIC.md);
- [boundary geometry](../../experimental/cancellation/BOUNDARY_GEOMETRY.md);
- [rigidity within the current ansatz](../../experimental/cancellation/RIGIDITY.md);
- [open problems](../../experimental/cancellation/OPEN_PROBLEMS.md).

Verification scripts remain active in the top-level `scripts/` directory.
