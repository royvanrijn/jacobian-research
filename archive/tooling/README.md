# Archived exploratory tooling

This directory contains search programs, bounded scans, diagnostic
enumerations, homotopy experiments, and generators that are not required by
the active proof suite.  They are preserved for provenance and future
experiments, but are intentionally absent from the main navigation and default
reproduction commands.

The archived three-real GMC programs reproduce bounded weighted-family
exclusions obtained before the explicit five-term witness settled every
dimension `n>=3`.  They are not candidates for extension; current Gaussian
work is restricted to GMC(2).
