# Naskręcki “Six Questions” provenance note

On 22 July 2026 the repository received a summary attributed to a recent
paper of Bartosz Naskręcki, but did not locate a complete public source,
bibliographic identifier, version, or stable URL.  The repository therefore
does not use that summary as an attribution source or as external review.

The theorem-by-theorem reconstruction made during that audit has since been
superseded by active documents and the central [status ledger](../STATUS.md).
The dated snapshot remains available only for provenance in the
[archive](../archive/provenance-audits/NASKRECKI_SIX_QUESTIONS_AUDIT.md).

If the underlying manuscript becomes available, compare its exact statements,
conventions, and certificate data with the independent repository proofs
before updating attribution or review status.
